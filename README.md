# Tyre Shop Management System

## 🚀 Quick Start

1. Install dependencies:
   ```
   npm install
   ```

2. Start the server:
   ```
   npm start
   ```

3. Open http://localhost:3000 in browser.

4. **Login**: username `admin`, password `1234`

## 📱 Features
- Modern responsive dashboard with sidebar
- Billing with WhatsApp share (exact format)
- Inventory with low stock alerts (<5 qty red)
- Expenses tracking
- Sales reports with date filter
- Customer search with service history
- Data persists in JSON files
- Print bills
